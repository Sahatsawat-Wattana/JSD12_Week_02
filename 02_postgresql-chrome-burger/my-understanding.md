postgres sql is one of open-source relational database management system. Sql is the structure query language that used to query for relational databses that manage data in tables in rows and columns. Each table has a required primary key and optional foreign keys that connect with other tables. The tables in relational databases require normalization to reduce the risk of error. Basic sql syntax is SELECT, INSERT, UPDATE, DELETE, FROM, WHERE, GROUP BY, ORDER BY, LIMIT, HAVING.
